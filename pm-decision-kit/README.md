# pm-decision-kit (plugin)

Decision-grade PM skills: specs, roadmaps, metrics, research synthesis, competitive briefs, stakeholder comms, assumption mapping, red-teaming, and a chained `/launch-decision` workflow — all driven by a single pluggable company-context file.

See the [repo README](../README.md) for installation, the context-file setup, and the demo.

**Skills (9):** operating-context (shared foundation), write-spec, roadmap-update, metrics-review, synthesize-research, competitive-brief, stakeholder-update, assumption-map, red-team

**Commands (1):** /launch-decision — chains metrics → research → assumptions → red-team → recommendation + comms
